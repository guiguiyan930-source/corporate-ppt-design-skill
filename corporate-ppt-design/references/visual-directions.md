# Visual Directions

只选一个主方向。品牌规范存在时，从品牌系统延展，不另起一套风格。

## 科技蓝白

- 适用：AI、SaaS、数字化、企业服务。
- 基调：理性、清洁、可信、轻盈。
- 配色：`#1E5BC6` 主色，`#2F80ED` 强调，`#A9C9FF` 辅助，`#F4F8FF` 背景，`#1F2937` 文字。
- 图片：服务器、芯片、数据网络、真实办公与产品场景。
- 版式：白底、大留白、局部蓝色结构线；避免整页浅蓝卡片。

## 深蓝发布会

- 适用：新品、硬件、汽车、能源、科技大会。
- 基调：沉稳、震撼、未来。
- 配色：`#061426` 背景，`#00C8FF` 强调，`#163A5F` 辅助，白色文字。
- 图片：真实产品、夜景、工业装置、山脉或能源场景。
- 版式：全屏大图、宽幅产品展示、大标题；避免过度光效。

## 黑底工业

- 适用：制造、物流、供应链、工业科技。
- 基调：力量、效率、可靠。
- 配色：`#05070A` 背景，`#1557FF` 主色，`#1B2433` 辅助，`#F2F5F9` 文字。
- 图片：工厂、机械臂、码头、集装箱、设备细节。
- 版式：大图与大数字，压缩型英文作辅助；不要卡通图标。

## 红白创新

- 适用：制造科技、汽车、能源、品牌升级。
- 基调：主动、现代、品牌鲜明。
- 配色：`#D71920` 主色，`#F3D5D8` 辅助，`#F7F8FA` 背景，`#202124` 文字。
- 图片：现代建筑、产品、研发与制造场景。
- 版式：白底、强对比、红色只用于关键锚点。

## 高级紫

- 适用：智慧办公、园区、地产科技、高端数字产品。
- 基调：精致、未来、克制。
- 配色：`#5B1FA6` 主色，`#6D3FD1` 强调，`#D8C8FF` 辅助，`#F8F5FF` 背景。
- 图片：玻璃建筑、园区、空间与设备，统一紫色环境光。
- 版式：图文分区、少量紫色标签；避免紫色渐变铺满全套。

## 奶油金品牌

- 适用：市场布局、品牌故事、规模与历程。
- 基调：温润、成熟、轻奢。
- 配色：`#B18A45` 主色，`#E8DCC5` 辅助，`#F7F3EA` 背景，`#292723` 文字。
- 图片：建筑、城市、地图、历史档案，暖色统一调色。
- 版式：地图、大数字、细线时间轴；不用于重技术参数页。

## 字体与层级

优先使用环境中可用且兼容交付平台的字体：

- 中文标题：思源黑体 / 阿里巴巴普惠体 / HarmonyOS Sans，Bold 或 Heavy。
- 中文正文：同字体 Regular。
- 英文：Inter / Arial / Helvetica Neue。
- 数字：DIN / Oswald；缺失时用英文主字体 Bold。

默认字号：

| 角色 | 字号 |
| --- | --- |
| 封面标题 | 50-72 pt |
| 章节标题 | 40-56 pt |
| 页面标题 | 35-44 pt |
| 小标题 | 24-28 pt |
| 正文 | 16-20 pt |
| 注释 | 10-12 pt |
| 大数字 | 60-120 pt |

## 图片提示词公式

```text
[行业真实场景/主体]，用于企业演示文稿，[镜头与构图]，
[主体位于左/右/中央]，[另一侧保留干净文字安全区]，
[材质与光线]，[视觉方向的色调]，真实摄影，专业商业质感，
16:9，高分辨率，无文字，无数字，无Logo，无水印
```

示例：

```text
现代智能制造车间与机械臂，用于工业科技企业介绍，
广角低机位，主体位于画面右侧，左侧保留深色干净文字安全区，
金属材质清晰，冷白工作灯与少量蓝色状态灯，黑底工业发布会色调，
真实摄影，专业商业质感，16:9，高分辨率，无文字，无数字，无Logo，无水印
```

## 反向约束

避免：低端模板感、卡通插画、无意义人物握手、廉价 3D 图标、花哨渐变、过度发光、颜色冲突、假文字、Logo 变形、密集装饰、所有元素居中堆叠、主体被裁掉、没有文字安全区。
